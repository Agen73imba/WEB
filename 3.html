JavaScript

Javascritpt JavaScript often abbreviated as JS, is a programming language that conforms to the ECMAScript specification. JavaScript is one of the core technologies of the World Wide Web. JavaScript enables interactive web pages and is an essential part of web applications. The vast majority of websites use it for client-side page behavior, and all major web browsers have a dedicated JavaScript engine to execute it.
